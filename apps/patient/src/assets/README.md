# Patient app assets

Replace the three icon files before submitting:

| File | Spec |
|---|---|
| `icon.png` | 1024×1024 PNG, opaque. Used by iOS App Store + general fallback. |
| `adaptive-icon.png` | 1024×1024 PNG, transparent foreground. Centred logo with ~20% padding (Android adaptive icon system crops aggressively). |
| `splash.png` | 1284×2778 PNG (iPhone 14 Pro Max size — Expo auto-scales). Centre the logo, transparent surround. Background colour defined in `app.json` as `#020617` (slate-950). |

**Brand palette (use these exactly):**
- Primary teal: `#14b8a6`
- Accent emerald: `#10b981`
- Background slate: `#020617`

Generate from your logo at https://www.canva.com or any design tool. The icon should be the OduDoc green-cross monogram, not the wordmark.
