20. Role Hierarchy

Super → platform operator; master access across tenants.

Admin → tenant (hospital) owner; full access within their organization.

Manager / Staff → scoped administrative operations (admissions, billing, inventory, etc.).

HR → staff, scheduling, careers, credentialing.

Support → tickets, feedback, notifications.

Doctor → own consultations, prescriptions, earnings, reviews.

Pharmacist → dispensing, pharmacy inventory, formulary.

Vendor → own products, orders, payouts.

Patient → own appointments, prescriptions, orders, health records.

21. Technical Foundation

Framework: Next.js 14 (App Router) · TypeScript · React Server Components.

Auth: NextAuth with credentials + JWT sessions; role-based middleware guards.

Database: PostgreSQL via Drizzle ORM.

Storage: Vercel Blob for documents and images.

Video: Daily.co integration with demo-mode fallback.

Presence & Signaling: In-memory TTL map with 5-second polling heartbeat.

AI Providers: Google Gemini (free tier), OpenAI, Anthropic Claude — precedence-based fallback.

Email / SMS: Transactional messaging via configurable SMTP and SMS provider.

Deployment: Vercel with serverless functions; environment-scoped configuration.
