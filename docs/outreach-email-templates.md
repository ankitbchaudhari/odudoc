# Doctor outreach email templates

Drop-in templates for `/admin/doctor-invites`. Each one is
specialty-specific so the recipient feels you actually read their
profile. Replace the bracketed bits with the doctor's name and any
unique credential you want to call out.

The /admin/doctor-invites page already has an automated generic
template. Use these when you want to **personalise individually** —
paste the body into your email client (Gmail, Outlook), edit the
square brackets, send.

## How to use these

The /admin/doctor-invites page sends the **generic template** (template 9) automatically with light personalisation when you paste a name + specialty. For higher-touch outreach to specific named doctors:

1. Pick the matching specialty template from above
2. Paste it into your normal email client (Gmail / Outlook)
3. Replace the bracketed bits — usually 2-3 substitutions per email
4. Send

After sending the personal version manually, **also** add the email to /admin/doctor-invites with note "personalised — [date]". This way the conversion column still updates when they sign up, and you keep an attribution trail.
